HTML Básico

HTML (HyperText Markup Language) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.

HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido.

<!DOCTYPE html> <html> <head> <title>Mi página</title> </head> <body> <h1>Bienvenido</h1> </body> </html>
  • HTML utiliza "marcado" para anotar texto, imágenes, y otros contenidos para ser mostrados en un navegador web.
  • HTML incluye elementos especiales como <head>, <title>, <body>, <header>, <footer>, <article>, <section>, entre otros.

Un elemento HTML se distingue de otro texto en un documento mediante "etiquetas", que consisten en el nombre del elemento rodeado por "<" y ">".

CSS Básico

CSS (Cascading Style Sheets) es el lenguaje que usamos para dar estilo a un documento HTML.

CSS describe cómo los elementos HTML deben ser mostrados en pantalla, papel, o en otros medios.

body { background-color: #f0f0f0; font-family: Arial, sans-serif; } h1 { color: blue; text-align: center; }
  • CSS se puede añadir a documentos HTML de tres maneras diferentes: en línea, interna y externa.
  • La forma más común y recomendada es un archivo CSS externo vinculado al HTML.

CSS utiliza selectores para aplicar estilos a elementos específicos en el documento HTML.

JavaScript Básico

JavaScript es un lenguaje de programación que permite implementar funcionalidades complejas en páginas web.

Cada vez que una página web hace más que simplemente mostrar información estática, es probable que JavaScript esté involucrado.

function saludar() { let nombre = document.getElementById("nombre").value; alert("¡Hola, " + nombre + "!"); }
  • JavaScript es un lenguaje de programación que te permite crear contenido que se actualiza dinámicamente.
  • Puede manipular el contenido HTML y CSS para actualizar una interfaz de usuario dinámicamente.

JavaScript es un lenguaje de programación completo, que puede ser utilizado tanto en el cliente como en el servidor con Node.js.

Responsive Design

El diseño responsive es una técnica de diseño web que hace que tu contenido web se vea bien en todos los dispositivos.

Un diseño responsive utiliza CSS y HTML para redimensionar, ocultar, reducir, ampliar o mover el contenido para que se vea bien en cualquier pantalla.

@media (max-width: 768px) { body { font-size: 16px; } .container { width: 100%; } }
  • Las media queries en CSS permiten aplicar estilos diferentes para diferentes tamaños de pantalla.
  • El viewport meta tag es esencial para un diseño responsive en dispositivos móviles.

Un enfoque común es diseñar para dispositivos móviles primero, luego agregar más estilos para pantallas más grandes.

Accesibilidad Web

La accesibilidad web significa que los sitios web, herramientas y tecnologías están diseñados y desarrollados para que las personas con discapacidades puedan usarlos.

Hacer tu sitio web accesible beneficia a individuos, negocios y la sociedad.

<img src="imagen.jpg" alt="Descripción de la imagen para lectores de pantalla"> <button aria-label="Cerrar" onclick="cerrarModal()">X</button>
  • Utilizar atributos como 'alt' en imágenes y 'aria-label' en elementos interactivos mejora la accesibilidad.
  • El contraste adecuado entre texto y fondo es importante para usuarios con deficiencias visuales.

Los estándares WCAG (Web Content Accessibility Guidelines) proporcionan pautas sobre cómo hacer que el contenido web sea más accesible.
